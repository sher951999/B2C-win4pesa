Server FAQ
Frequently Asked Questions About Server

Do I need VPS or a Dedicated server to run this system?

No, You can run this system in a shared host as well and it will run smoothly. But if you have much traffic, you may need to upgrade your server/hosting. Our codes are highly optimized to ensure the best possible performance.

Which hosting provider should I choose for this? Do you have any suggestions?

You can choose any hosting provider that provides cPanel-based hosting. cPanel is not a must, but we recommend you to have cPanel-based hosting.

I don't have cPanel, What should I do?

Nothing to worry about, Our system will work with any control panel but cPanel is easy to manage, for that we recommend cPanel-based hosting. Also, Our free Support is limited to cPanel-based hosting only.

Do you provide hosting as well?

Yes, We do. We can provide a complete solution for your hosting. We can provide budget shared hosting, premium shared hosting, VPS, Dedicated servers, and Cluster from various geolocation. Please contact us for more details.

Admin Dashboard
Dashboard Overview

The Admin Dashboard menu in BetLab offers a comprehensive set of widgets to empower administrators with essential insights and statistics. It provides a quick overview of the platform's key metrics, including the number of bettors, pending bets, deposits, withdrawals, and support tickets. Additionally, administrators can track the total deposited and withdrawn amounts, along with associated charges. Interactive monthly deposit and withdrawal charts and a transaction chart enable better financial planning and analysis. This robust Admin Dashboard menu streamlines platform management, ensuring seamless operations and exceptional user experiences

You will encounter a popup window containing important instructions for configuring the cron job. It is essential to set up the cron job commands provided in that window on your server. By doing so, you can automate the return process of placed bets, ensuring a smooth and efficient betting experience for your users.

Manage Bettors
Mange Users Overview

The "Manage Bettors" menu in BetLab's Admin Dashboard offers comprehensive tools for effective user management. Admins can access lists of all bettors, active and banned bettors, as well as those with pending KYC verification.

The menu includes options to send notifications to all bettors and provides a link to detailed bettor insights on betting, payments, and transactions. Additionally, administrators can update bettor profile data for personalized support and a secure betting environment.

On the details page within the "Manage Bettors" menu, administrators have an overview of the bettor's essential information, including their account balance, deposited and withdrawn amounts, as well as the total amount bet on placed bets.

Admins hold the authority to update bettor information, allowing them to make necessary changes and adjustments. They can also add or subtract from the bettor's balance as required, ensuring efficient fund management.

Furthermore, from the details page, administrators can view the bettor's login history, enabling a comprehensive understanding of their account activity. Additionally, admins have the capability to ban bettors directly from this page, maintaining control over the platform's integrity and security.

Email Verification: If a user's email verification is unverified, a verification code will be sent to their email when they attempt to log in. They must verify their account using the code for enhanced security.

Mobile Verification: Similarly, if a user's mobile verification is unverified, a verification code will be sent to their mobile when they attempt to log in. They must verify their account using the code for added security.

2FA Verification: If a user enables 2FA security from their user panel, they need to verify the code generated by their Google Authenticator app to gain access to their account.

KYC: If a user's KYC status is unverified, they need to submit the necessary information as configured in the KYC Setting to comply with regulatory requirements.

Manage Games
Mange Users Overview

From this menu admin can add or update all games for bet

Ended Games: The list of games for which the end time has already passed the current time. This feature enables administrators to quickly identify and review completed games, providing essential data and statistics for further analysis and reporting.

Upcoming Games: The list of games where the betting period has not yet started. This feature allows administrators to easily identify and manage upcoming events, enabling them to prepare the platform for bettors to place their wagers when the betting window opens.

Running Games: The list of games where the betting period has started but has not yet ended. This feature allows administrators to monitor ongoing events.

How to add new games?

To add a new game, simply navigate to the "Manage Games" menu and click on the "Add New" button under the "All" submenu.

How to add markets, option/selections & odd rates?

From the games list, administrators have the flexibility to add markets to a specific game. Additionally, options or selections can also be added directly from this page. This convenient feature allows administrators to set odd rates for each selection while adding them. By providing the ability to define odds, administrators can customize and optimize the betting experience, ensuring competitive and enticing betting options for users to engage with. This comprehensive functionality empowers administrators to manage betting markets efficiently and fine-tune odds to create an engaging and dynamic platform for sports betting enthusiasts.

Note: The odds rate should be entered in decimal format.
